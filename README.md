# Grimoire Extensions

Public content-source extensions for the [Grimoire](https://grimoireapp.org)
reader app. Each extension is a small Android APK that the Grimoire app loads
at runtime.

## Add this repo to Grimoire

**One-tap on Android (recommended):**

[**Add Grimoire Extensions to the app →**](https://grimoireapp.org/add-repo?name=Grimoire%20Extensions&url=https%3A%2F%2Fgithub.com%2FOperation-Grimoire%2Fgrimoire-extensions%2Freleases%2Fdownload%2Flatest%2Findex.json)

If Grimoire is installed, the link opens the app with the "Add Repository"
dialog pre-filled — just tap **Save**. If not, you'll land on
[grimoireapp.org](https://grimoireapp.org) with a download prompt.

**Manual fallback** — use this if the magic link doesn't open the app, or if
you'd rather type things in yourself:

1. Open Grimoire → **Extensions** (top-right `▤` icon) → **+** to add a
   repository.
2. Fill in:
   - **Name:** `Grimoire Extensions`
   - **Index URL:** `https://github.com/Operation-Grimoire/grimoire-extensions/releases/download/latest/index.json`
3. Tap **Save**. Available extensions appear in the list within a few
   seconds.
